비트 단위로 연산이 진행되는 연산자

### 비트연산자 사용법
`<<` `>>`

* 비트를 화살표가 가리키는 방향으로 한 칸씩 민다.
* 비트가 한 자리 올라가는 것은 `2배`가 된다는 의미다.


```cpp
unsigned char byte = 13;
byte <<= 3;  // 2^n 배수
byte >>= 1;  // 2^ 나눈 몫
```


|    |                             |
|----|-----------------------------|
| &  | 비트 곱: 둘 다 1인 경우 1     |
| \| | 합 : 둘 중 하나라도 1이면 1   |
| ^  | xor : 같으면 0, 다르면 1      |
| ~  | 반전 : 1은 0으로, 0은 1로     |


* 비트 빼는 구문 눈에 익숙하게 외워두기!
* 상대적으로 비중이 적은 연산자이다.
* **동작방식**을 기억하기!
