## 변수의 종류  

|---------------|-----------------------------|
| **지역변수**  |  함수 **내부**에 선언된 변수<br/>해당 함수의 스택 메모리안에 들어있다.  |
| **전역변수**  |  함수 **외부**에 선언된 변수<br/>Data영역을 사용한다.  |
| **정적변수**  |  Data영역을 사용한다.  |
| **외부변수**  |  Data영역을 사용한다.  |

<br/>

### Data 영역 특징
* 프로그램 시작 시 생성
* 프로그램 종료 시 해제


## 메모리 영역  

| **스택 영역**              |
| **데이터 영역**            |
| **읽기 전용(코드, ROM)**   |
| **힙 영역**               |


<br/>

```cpp
#include <stdio.h>

// 전역변수
int g_i = 0; // Data 영역

// Data 영역 특징
// 프로그램 시작 시 생성
// 프로그램 종료 시 해제


void Test()
{
	++g_i;
}


int main()
{
	int a = 0;
	Test();
	Test();
	Test();


	return 0;
}
```
