## 1. DFS 기본 개념  
깊이 우선 탐색. 
시간복잡도는 O(V+E) V:정점, E:간선  

<br>


## 2. 자주 쓰는 패턴  

**(1) 기본 뼈대**  
```c++
vector<int>v [1004];
bool visited[1004];

void dfs(int here)
{
    visited[here] = true;

    for(int i : v[here])
    {
        if(!visited[i])
        {
            dfs(i);
        }
    }
}

```
