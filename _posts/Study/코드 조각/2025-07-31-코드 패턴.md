## 정수 뒤집기  
예: `num = 1234` 라면, `reverse = 4321`이 되도록 만드는 게 목표  

```cpp
int reverse = 0;
while (num > 0) {
    reverse = reverse * 10 + (num % 10);
    num /= 10;
}
```

`% 10` → 맨 마지막 자릿수 가져오기  
`reverse * 10 + (마지막 자릿수)` → 숫자를 왼쪽으로 밀고 자릿수 추가  
`/= 10` → 맨 마지막 자릿수 제거
