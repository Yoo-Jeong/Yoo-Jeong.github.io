```cpp
data = 0;

++data;  // 전위(전치)
data++;  // 후위(후치)

--data;
data--;
```
* 후위 연산자로 사용하는 경우, 연산자 우선순위가 **가장 나중**으로 밀린다.  


특별한 사유가 없다면 **전위로 사용**하는 습관을 들이자. (실수방지)
