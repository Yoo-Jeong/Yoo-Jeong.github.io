## 산술 연산자  
```
+
-
*
/
% (모듈러스, 나머지)
```

```
++
--
```


```cpp
int data = 1 + 1;
data = data + 20; 
//data가 두번 반복된다. 이를 편하게 하기 위해서 ↓
data += 20;
```

<br>

## 연산자 우선순위
연산자들이 나열됐을 때 우선순위가 있다.    

<br>

## 나머지 연산자 `%`
 
 
실수끼리의 나머지 연산(%)이 가능한가? 나머지 개념이 존재하는가?  
 👉 *불가능하다.*  


따라서 **나머지 연산자(%)는 피연산자가 모두 정수일 경우에만 사용**할 수 있는 개념이다.
